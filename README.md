Java Wrapper API for HackerNews

# How to use

### Gather the posts from a desired page
	Post[] posts = Page.get(#)	//Gather an array of the posts from the desired page

### Gather information on those posts
	posts[1].getTitle();

At the moment, you can get the:

* Rank
* Title
* Link URL
* Points
* User
* User Profile URL
* Posted Time
* Number of comments
* Comment URL
